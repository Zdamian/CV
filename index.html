About me

Witam. Nazywam się Damian. Moją najwiekszą pasją jest rysowanie, które stało się częścią mojego życia. Z czasem zacząłem interesować się grafiką komputerową. Pierwsze kroki związane z grafikę stawiałem w gimpie a następnie w Blender od wersji 2.6. Poznałem także podstawy w programie VUE 11. Ostanio zacząłem się zajmować tworzeniem stron internetowych oraz pisaniem aplikacji, ciągle pogłębiając wiedzę w tej dziedzinie.

Hello. My name is Damian. My greatest passion is drawing which became part of my life. With time I started interested in computer graphics. I had begun learning in software Gimp and next in Blender from version 2.6. I also learned the basics of the program VUE 11. In last time I began create websites and writing web applications still broadening one’s knowledge.

About my works

Grafiką zajmuję się od małego. Będąc dzieckiem kochałem już rysować a z każdą pracą coraz lepiej mi to wychodziło. Do dnia dzisiejszego narysowałem około 200 rysunków. Wsród moich prac jest dużo postaci z ulubionych filmów, między innymi Transformer czy z uniwersum Marvela.

Gdy rysowanie nie dawało mi już tyle satysfakcji zacząłem robić prace 3D w Blender i pogłębiałem wiedzę o grafice komputerowej. Do tej pory zrobiłem kilka krótkich prostych animacji które umieściłem w portfolio wraz z moimi najlepszymi renderami.

Ostatnio zajmuję się tworzeniem stron internetowych. Do tej pory opanowałem HTML i CSS w stopniu dobrym oraz JavaScript oraz jQuery w stopniu średnim. Nauczyłem się także korzystać z Git oraz pisać testy do własnego kodu JavaScript. Podczas nauki napisałem aplikację Todo z wykorzystaniem modelu MVC oraz napisałem kilka templates w Bootsrapie. Nauczyłem się też pisać style w Sass w dobrym stopniu.

When I was child I loved already drawing and with each project I was better in this. Up to now I did about 200 drawings. Among my works is a lot of characters from my favorite movies, among others Transformers and from The universe of Marvel.

When drawing didn't give me much satisfaction I began do the 3D models in Blender software and I was broadening one's knowledge about computer graphics. Up to now I did a few short and simple animations which you can see in my portfolio with my best works made in Blender.

Recently I'm learning create websites. Up to now I master HTML and CSS in a good level, JavaScript and jQuery in a medium level. I also learned how to use Git and writing a tests in Jasmine to own code JavaScript. During my learning I wrote my first application "Todo" with using model MVC and a few templates written in Bootstrap. I also learned how write styles in SASS in a good level.
